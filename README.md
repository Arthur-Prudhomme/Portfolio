# Portfolio

## To install and use my project :
- download the zip
- extract it where you want
- open the `index.html` in your browser

## If you want to modify it :
- install sass
- open cmd
- type `cd <file path>`
- enter the file "styles" with the same command
- type the command `sass --watch style.scss:build.css`
- modify the style.scss

## How I deployed it :
To deploy my project I used Vercel by connecting my github account and choosing a repository to deploy and gitio
